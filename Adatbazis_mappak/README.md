# Adatbazis_mappak mappa

Ebben a mappában találhatóak a tanító és kiértékelő kód által is használt adatbázisok. Ezek természetesen a méretük miatt nem kerültek feltöltésre, ezt a felhasználónak kell pótolni kézzel. Értelemszerűen,
a cifar10 mappába helyezzük be a cifar10 adatbázist, a cifar100 mappába pedig a cifar100 adatbázist.

Ezek elérhetőek a következő címen: https://www.cs.toronto.edu/~kriz/cifar.html

Python scriptet is írhatunk az adatbázisok letöltéséhez, hiszen a torchvision.datasets package-ben is elérhetőek ezek.
